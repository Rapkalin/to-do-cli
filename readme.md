OBJECTIF
Créer un mini projet pour gérer sa TODO dans son Terminal. Le projet sera composé :
- D'une base de données en SQLITE
- D'un PROGRAMME en PHP qui permettra d'intéragir avec la base de données SQLITE
  Le projet pourra être versionné sur Github.

Un utilisateur pourra intéragir avec son Terminal pour :
- (CREATE) CRÉER/AJOUTER DES ITEMS
- (READ) CONSULTER DES ITEMS
- (UPDATE) MODIFIER DES ÉLÉMENTS
- (DELETE) SUPPRIMER DES ÉLÉMENTS

1- Sur papier, poser la structure de la base de données avec les tables qui la composeront
2- Sur papier, organiser la structure de notre programme et des fichiers (PHP, BDD etc.) qui le composeront
3- Créer la base de données SQLITE et les tables qui vont avec

5- Créer la seconde commande PHP pour consulter les items

7- Créer la dernière commande PHP pour supprimer les items
